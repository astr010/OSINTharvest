1. Overview
	OSINTharvest is a tool designed and created to assist in the gathering, parsing and review of 	open source intelligence from public sources. The output of OSINTharvest can be used as 	enrichment data for all SIEMs as indicators of compromise.


2. Modules
	Core:		The core module is used to navigate and launch the OSINTharvest modules.

	LISTharvest:	LISTharvest is used to gather open source blacklists, malware domains and 			entities seen to be involved in 				malicious 			activity.

	Simply navigate to the OSINTharvest director and enter:	> python core.py
